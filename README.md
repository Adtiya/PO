## ğŸš€ **Quick Start**

### **Prerequisites**
- Python 3.11+
- Node.js 20+
- PostgreSQL 14+
- Redis 6+
- Docker (optional)

### **1. Clone Repository**
```bash
git clone https://github.com/your-username/agi-nari-enterprise.git
cd agi-nari-enterprise
```

### **2. Environment Setup**
```bash
# Copy environment template
cp enterprise_system/.env.example enterprise_system/.env

# Edit configuration (required)
nano enterprise_system/.env
```

### **3. Install Dependencies**
```bash
# Backend dependencies
cd enterprise_system/backend
pip install -r requirements.txt

# Frontend dependencies
cd ../enterprise-ai-frontend
npm install

# Microservices dependencies
cd ../microservices
./install_all_dependencies.sh
```

### **4. Database Setup**
```bash
# Initialize database
cd ../backend
python run.py --init-db

# Run migrations
python run.py --migrate
```

### **5. Start Services**
```bash
# Start all services
python ../start_all_services.py

# Or start individually
python backend/run.py &
python microservices/ai_nlp_service/nlp_ai_service/src/main.py &
# ... other services
```

### **6. Access System**
- **Frontend**: http://localhost:3000
- **API Gateway**: http://localhost:6000
- **Health Checks**: http://localhost:8000/health
